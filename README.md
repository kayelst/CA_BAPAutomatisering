#Cloud Applications 2016-2017/ Bachelerproef Automatisering / Bryan Lameir & Kay Elst

##INFO:
We moesten voor deze opdracht het managen van de bachelorproef zoveel mogelijk automatiseren. 
Dit houdt in dat alles via "een druk op de knop" of automatisch uitgelezen en verzameld wordt in eenvoudig te gebruiken windows.
Aan de hand van deze info kan de leerkracht zijn studenten managen en volgen op een efficiënte manier.

##How To Run?
1) Clone de git naar volgende link op uw Computer: C:\Users\Username\Desktop\BapAutomatisering
2) Open run.bat
3) Ga in uw browser naar localhost:3000
